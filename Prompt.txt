# PROMPT – VizPM v2.01 (Visual Project Manager OS)

**Act as a Principal Full-Stack Software Engineer and UI/UX Architect.** Build a high-performance, OS-like web application called **VizPM**. It combines an infinite canvas, Kanban, Gantt, directed graphs, and SCADA monitoring. Ensure a **Single Source of Truth**: any state change instantly propagates across all views.

## 1. GLOBAL APP SHELL
- **Top Header**: "VizPM" logo, File Menu (New, Open, Save, Save Draft, Load Version), Global View Toggles (Canvas, Gantt, Kanban, Graph), clear Simple/Advanced toggle switch, User Avatar.
- **Bottom Taskbar**: Icons for open project windows (minimize, maximize, close).
- **Persistence**: `localStorage` for global settings, open windows, and activity logs.

## 2. PROJECT WINDOW (CANVAS AS `main`)
- Projects open as floating, resizable windows with a standard title bar: `[Project Name] [—] [□] [X]`.
- The infinite canvas is technically the root card (`id: "main"`, `parentId: null`). It holds all project properties and contains macro-blocks as direct children. Initial size: 8000x5000px (configurable).

## 3. CARD ANATOMY & HIERARCHY (v2.01)
Strict 3-level hierarchy: Macro-block → Component → Sub-component. Highly configurable via the Advanced Side Panel.
- **Collapsed Mode**: Shows **ONLY Zone A**. Everything else is hidden.
- **Compact Mode (`compactView: true`)**: Fixed 4:3 ratio. Full-cover background image with dark gradient overlay. Shows: truncated title (max 30 chars), mini avatars, mini progress bar, and "Quick view" toggle.
- **Detailed Mode Zones** (Default top-to-bottom, order/visibility configurable):
  - **Zone A (Unified Header + Progress)**: Always visible if not collapsed. A1: Dark bg, left accent border (`borderColor`), title (max 50 chars, ellipsis), collapse/expand chevron, "Quick view" eye icon. **NO icons, WBS, or % here**. A2: Thin full-width progress bar.
  - **Zone B (Main Media)**: Primary image (`image.src`). Configurable as top banner or left/right column.
  - **Zone C (Metadata)**: Owner avatars, Area tag, % text, Gear icon (opens side panel), IDEA bulb icon. *Advanced toggle*: "Compact Metadata" to shrink this to a single minimal line.
  - **Zone D (Tabs)**: "Work" (checklist + "Auto-complete" toggle), "Technical", "Economic", "Reqs", "Comments".
  - **Zone E (Child Dropzone)**: Container for child cards. Shows guide text if empty. Hidden if collapsed.
  - **Zone F (Optional Child Gallery)**: Advanced toggle to show a grid (`grid-cols-2` or `3`) of child cards' main images.
- **Layout Config**: Per-card `zonesOrder` (e.g., `["A", "C", "E", "D"]`), `hiddenZones` (e.g., `["B"]`), and `layoutOrientation` (force children in Zone E to `"vertical"` or `"horizontal"`).
- **Adaptive Scroll**: If resized, Zone A stays `sticky` (`top: 0`). Zones C-F get `overflow-y: auto` with hidden scrollbar (`no-scrollbar`).

## 4. SMART SIDE PANEL (TRIANGLE TOGGLE)
- A triangle button (▶/◀) is **always visible** on the right edge of the project window.
- **Auto-expands** when any card is clicked, showing deep-edit properties for that specific card.
- **Contents**: Title/desc inputs, all Tabs, and an **"Advanced Card Config"** section (Zone reordering/hiding, Compact Metadata toggle, Child Orientation toggle, Child Gallery toggle, Notification rules, Permissions, Change history).

## 5. REACTIVE VIEWS (SINGLE SOURCE OF TRUTH)
- **Canvas (SCADA)**: Infinite pan/zoom, GPU transforms, grid, animated SVG Bézier connections, minimap, owner filter (dims non-matching).
- **Gantt**: Root cards as horizontal bars. Drag/resize updates dates globally. Finish-to-Start dependencies highlight the critical path in red.
- **Kanban**: Columns (Not Started, In Progress, Blocked, Completed). Dragging changes `status` globally.
- **Graph**: Force-directed/hierarchical layout of all cards and connections. Expandable nodes.
- **Presentation Mode**: Smooth camera transitions between selected/root cards.

## 6. COLLAPSIBLE EXECUTIVE DASHBOARD
- Top panel inside the Project Window, fully collapsible via toggle to save vertical space.
- Real-time KPIs: Global % progress, Budget vs Spent (donut chart), Active blockers list, Critical path nodes, Next 5 milestones.

## 7. DATA INTELLIGENCE (ROLLUP & ALERTS)
- **Hierarchical Rollup**: Parent `presupuesto`/`gastado` = sum of descendants. Parent `progress` = weighted average of direct children. Aggregated values show a "∑" icon. Manual edit breaks aggregation (show warning).
- **Cascading Alerts**: Monitors subtrees for `status: 'Blocked'` or `reqs: 'pending'`. Bubbles up to the root with a pulsing red badge (`pulse-border`) in Zone A/C, even when collapsed.
- **Smart Notifications**: `notificationRules` engine (e.g., `trigger: "dueDate < 48h"`, `action: "badge + webhook"`).

## 8. PERFORMANCE & INTERACTIONS
- **Optimized Drag**: GPU-accelerated "ghost" proxy during drag. Real DOM updates only on `drop`.
- **Marquee Selection**: `requestAnimationFrame` + viewport culling for performance.
- **Smart Drop Zones**: Dragging over a card highlights 3 zones: Top edge (previous sibling), Center (child), Bottom edge (next sibling).
- **Shortcuts**: Ctrl+Z/Y (Undo/Redo, 50 steps), Ctrl+C/V (Duplicate with new ID/offset), Delete (with confirmation).

## 9. PERSISTENCE & VERSIONING
- **localStorage keys**: `vizpm_projects`, `vizpm_open_windows`, `vizpm_global_settings`, `vizpm_activity_log`, `undoStack`/`redoStack`.
- **Auto-save**: Every 5 mins + 2s debounce after significant actions (drop, resize end, status change).
- **Versions**: "Save as draft" creates a named snapshot. "Load version" restores with confirmation.
- **Persistent IDEA Notes**: Saved inside the card's JSON (`ideaNotes: [{text, timestamp}]`), ensuring they export/import with the project.

## 10. COLLABORATION PREP
- Data model includes `users: [{name, role}]` and optional `permissions` per card.
- State mutations structured as operations: `{ userId, timestamp, targetId, field, value }` (ready for future Yjs/CRDT and offline-first queueing).

## 11. VISUAL MODES
- **Simple / Advanced**: Global toggle. "Simple" hides WBS, rollup math, Graph view, and technical tabs.
- **Focus Modes**: "Work" (hides children, expands tabs), "Blocks" (hides tabs, expands children), "Hybrid" (all visible).

## 12. UI/UX CONSTRAINTS (TAILWIND + DAISYUI)
- **Stack**: HTML5, Vanilla ES6+ JS, **Tailwind CSS and DaisyUI via CDN** (mandatory for premium components and `data-theme="dark"`).
- **Theme**: Premium dark mode (Vercel/Shadcn style). Bgs `#0b0f19`, cards `#111827`, crisp white text, blue/purple/green accents.
- **Styling**: Consistent `border-radius`, 0.2s transitions, `object-fit: cover` for images.
- **Animations**: Smooth slide-in/slide-out for side panels, dashboard, and modals.

## 13. DELIVERABLES
Generate **ONE single, fully functional file** without placeholders:
1. **`index.html`**: Contains all HTML, CSS (Tailwind/DaisyUI CDN), and Vanilla ES6+ JS implementing **every point above** with precision.

This file contains the "AeroVend 1.0 v2.01" project (June 1, 2026 start, July 3, 2026 Uniovi Emprende deadline, 7 macro-blocks, 3-level hierarchy, all v2.01 fields like `zonesOrder`, `hiddenZones`, `layoutOrientation`, `ideaNotes`, and realistic team data).
